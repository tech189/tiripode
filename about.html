About Tiripode

This website has a set of tools built for those studying Mycenaean Greek / Linear B. I built these partly because I wanted something similar to exist while I was studying the language myself. It was built as part of my Computer Science MSc final project.


Parsable words

About 70% of the full lexicon is parsable. These were generated based on the assumption that the lexicon contains mostly nominative headwords ‚Äî they may not be correct, so please use with caution (e.g. a 3rd declension ending -a may have been interpreted as a 1st declension nominative and added to the list). The oblique cases are parsable but are not included in the list below.
